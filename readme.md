# 🍔 HBurguer

Landing page moderna para hamburgueria desenvolvida com HTML, CSS e JavaScript puro.

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- JSON Fake API
- Glassmorphism UI
- Responsive Design

---

## ✨ Funcionalidades

✅ Layout responsivo  
✅ Navbar premium  
✅ Spotlight effect  
✅ FAQ expansível  
✅ Modal de produtos  
✅ Cards dinâmicos  
✅ Integração com WhatsApp  
✅ Formulário com validação  
✅ Carrossel automático  
✅ Filtro de cardápio  
✅ Consumo de API fake JSON  
✅ Animações modernas
